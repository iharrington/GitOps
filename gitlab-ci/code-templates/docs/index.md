# Code Templates

Welcome to the **Code Templates** documentation.

This repository provides reusable boilerplate files and configuration templates
for common development and DevOps scenarios.

## Current Templates

- [.gitignore Templates](gitignore.md) — for Terraform/Terragrunt, Helm/Kubernetes, Node/NPM, and general development.
- [.dockerignore Template](dockerignore.md) — standard Docker ignore patterns for Docker projects.
- [.helmignore Template](helmignore.md) — standard Helm ignore patterns for Helm & Helm-Chart projects.

## Why Templates?

- Standardize configuration across projects
- Reduce repetitive setup work
- Encourage best practices and consistency
